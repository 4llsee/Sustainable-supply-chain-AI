# Sustainable-supply-chain-AI
AI-powered system for predicting carbon emissions and generating intelligent sustainability recommendations for supply chains.

## Project Overview
This graduation project combines Machine Learning and Large Language Models (LLMs) to help organizations analyze environmental impact and improve sustainability in supply chain operations.

The system predicts carbon emissions using machine learning models and generates smart recommendations through an integrated LLM-based solution.

## Features
- Carbon emission prediction
- Data preprocessing and feature engineering
- Machine learning model training and evaluation
- AI-generated sustainability recommendations
- Multi-dataset integration and analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning
- Large Language Models (LLMs)

## Project Goal
Building intelligent AI solutions that support sustainable decision-making and reduce environmental impact in supply chains.
